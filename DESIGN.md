### APIs

| API | Use | Link |
| --- | --- | --- |
| Food Tracker | User can look up all kinds of food and their nutritional value | https://www.foodrepo.org/ch/products |
| Comparing Results | Compares completed fasts by other users | https://ide50-farginda.cs50.io:8080/list |

De _Food Tracker_ API zou ik in de app willen gebruiken voor extra informatie over de voedingswaarde van verschillende soorten voedsel. De _Comparing Results_ API wordt gebruikt om de resultaten van verschillende gebruikers te vergelijken na het beindingen van de _fast_.

Verder zou ik een extra _data source_ willen toevoegen die meer informatie kan bieden over het idee van _intermittent fasting_ via de volgende links: 
* https://www.healthline.com/nutrition/10-health-benefits-of-intermittent-fasting
* http://time.com/5354498/is-intermittent-fasting-healthy/
* https://www.health.harvard.edu/blog/intermittent-fasting-surprising-update-2018062914156
* https://www.mindbodygreen.com/articles/why-intermittent-fasting-is-the-best-thing-to-ever-happen-to-your-metabolism
* https://www.dietdoctor.com/intermittent-fasting
